# SportsStream Pro - Vista final

Prototipo visual final de la plataforma web. Abre `index.html` para ver como quedaria la web.

Esta version esta pensada para que luego el frontend consuma una API compartida con una futura app movil.

## Reproductor real

La seccion del reproductor permite pegar una URL HLS (.m3u8) o MP4 propia/autorizada. Tambien incluye un boton "Probar demo HLS" con una fuente publica de prueba.

No uses enlaces de sitios de terceros si no tienes permiso para retransmitirlos.
